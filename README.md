# Indian Monsoon Low Atlas

Static, self-contained atlas for the ERA5-derived LPS v5.2 South Asian low-pressure-system track catalogue.

Coverage in the embedded snapshot is 1940-03-26 16:00 UTC through 2026-01-10 08:00 UTC. Default seasonal views use complete years through 2025; 2026 is treated as partial coverage rather than a zero-count season.

Notes:

- Peak classes are ERA5-derived IMD-equivalent classes, not official IMD classifications.
- State rainfall diagnostics are concurrent state-mean rainfall while a system exists, not storm-attributable rainfall.
- Headline records exclude tracks with major gap, low-coverage, or large-jump quality flags by default.
- The app is bundled as a single HTML file with embedded compressed catalogue and map payloads.
